# Мои ремонты

Устанавливаемое приложение для Samsung/Android. Хранит заявки на телефоне и отправляет их в Google Таблицу.

## Формула топлива

`расстояние до клиента × 2 × 7 / 100 × 1,60 €`

Прибыль считается как `стоимость ремонта − комплектующие − бензин`.

## Подключение Google Таблицы

Таблица остаётся приватной. В приложении нужно войти как `service.rollershutter@gmail.com`. После этого кнопка с облаком добавляет новую строку или обновляет существующую заявку через Google Sheets API.

Адрес таблицы и идентификатор входа настроены внутри приложения. Секретные ключи и общедоступный Apps Script не используются.

## Установка на Samsung

Откройте `https://oblakobezshtanov.github.io/moi-remonty/` в Chrome и выберите «Добавить на главный экран» или «Установить приложение».
